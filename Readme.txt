FrontEnd -

App.jsx -  Main component handles state for amount, currency, and result

UseState - For handling the states of error

App.css - for styling up the webpage


Backend - 

Server.js - A server with a /convert endpoint

express.JSON() - handles post request bodies

result -  The Converted Value,returns as the answer.


To run the FrontEnd-

-->cd FrontEnd
-->npm install 
-->npm run dev

-->The Frontend server is running on port number 5173

To run BackEnd -

-->cd Server(filename)
-->npm install 
-->node server.js

-->The backend server is running on port number 3000

Note-

Incase if it's needed to run locally,We can change the port numbers near api calls.
